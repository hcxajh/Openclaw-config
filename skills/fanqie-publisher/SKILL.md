# 番茄小说短故事发布

## 快速开始

1. 准备故事文件（txt格式）和封面图片（png格式）
2. 使用 browser 工具打开发布页面
3. 按步骤自动化填写表单

## 发布流程

### 1. 打开发布页面
```
URL: https://fanqienovel.com/main/writer/publish-short
```

### 2. 输入标题
- 使用 `browser act` + `kind: type` 输入标题
- 如果标题重复，自动添加后缀（如"-新版"）

### 3. 输入正文
- 使用 JavaScript evaluate 注入内容
- **重要**：每次注入不超过5000字，较长内容需分段注入
```javascript
document.querySelector('[contenteditable="true"]').innerHTML = '<p>故事内容</p>'
```

### 4. 设置AI使用
- 点击"是"单选框：`ref=e73`

### 5. 设置分类
- 点击分类选择器：`ref=e83`
- 选择"社会伦理"：`ref=e200`
- 按 Escape 关闭弹窗

### 6. 设置试读比例
- 点击"去设置"按钮
- 系统自动设置为30-32%

### 7. 勾选发布协议
- 点击复选框：`ref=e107`

### 8. 点击下一步
- 如果提示"暂未设置章节"，需要启用**智能分章**开关
- 使用 JS evaluate 启用：`document.querySelector('switch').click()`
- 继续点击下一步

### 9. 封面设置（可选）
- 方案A：使用模板（推荐）
  - 点击"封面制作" → 选择模板 → "完成制作"
- 方案B：AI生成
- 方案C：手动上传（需用户操作）

## 自动化难点

| 步骤 | 难点 | 解决方案 |
|------|------|----------|
| 正文输入 | JS注入<5000字，剪贴板粘贴被阻止 | 让用户手动粘贴完整内容 |
| 封面上传 | 反自动化保护 | 使用封面模板或AI生成 |
| 弹窗处理 | ref每次刷新会变 | 截图确认后操作 |

## 相关文件

- 故事文件：`fanqie/日期/序号/故事名.txt`
- 封面文件：`fanqie/日期/序号/*.png`
- 详细流程：见 [references/workflow.md](references/workflow.md)

用户说"发布003文件夹的故事"，操作步骤：
